# UI/UX Wireframes for Key Pages

## Home Page
- **Header**: Logo, Navigation (Home, Menu, Cart, Profile), Login/Register buttons
- **Hero Section**: Welcome message, call-to-action button "Order Now"
- **Featured Menu**: Carousel of popular items with images, names, prices
- **Footer**: Contact info, social links, copyright

Layout: Responsive grid, mobile-friendly with hamburger menu.

## Menu Page
- **Filters/Sidebar**: Categories (e.g., Appetizers, Main Course), Search bar
- **Menu Grid**: Cards with item image, name, description, price, "Add to Cart" button
- **Pagination**: For large menus

Layout: Grid layout, filters on left, items on right.

## Order/Cart Page
- **Cart Items List**: Each item with quantity selector, remove button, subtotal
- **Order Summary**: Total, taxes, delivery fee
- **Checkout Section**: Payment method selection, place order button
- **Empty Cart Message**: If no items

Layout: Two-column, items on left, summary on right.

## Profile Page
- **User Info Section**: Editable fields for name, email, phone, address
- **Order History**: List of past orders with status, date, view details
- **Settings**: Change password, notifications preferences

Layout: Tabs for sections, form inputs for editing.

## Admin Dashboard
- **Stats Overview**: Cards for total orders, revenue, active users
- **Manage Menu**: Table of menu items with edit/delete actions
- **Manage Orders**: Table of orders with status update dropdowns
- **Navigation Sidebar**: Links to different admin sections

Layout: Dashboard with sidebar, main content area with tables and charts.

All pages use consistent color scheme (e.g., green for food theme), accessible fonts, and touch-friendly buttons for mobile.
